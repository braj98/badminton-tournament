Step 1 (Do First)

Move ALL CSS.

Create:

css/styles.css

Copy everything inside:

<style>
...
</style>

into:

styles.css

Then replace with:

<link rel="stylesheet" href="css/styles.css">

This alone removes hundreds of lines.

app.js < 300 lines
Important

Do NOT use ES6 modules yet.

Avoid:

import ...
export ...

for now.

Use simple script tags:

<script src="js/state.js"></script>
<script src="js/supabase.js"></script>
<script src="js/categories.js"></script>
<script src="js/setup.js"></script>
<script src="js/groups.js"></script>
<script src="js/fixtures.js"></script>
<script src="js/standings.js"></script>
<script src="js/knockout.js"></script>
<script src="js/champion.js"></script>
<script src="js/app.js"></script>

This works perfectly on GitHub Pages and avoids CORS/module headaches.

My recommendation

Don't try to split the whole file in one go.

Do it in this order:

1. styles.css
2. supabase.js
3. state.js
4. setup.js
5. groups.js
6. fixtures.js
7. standings.js
8. knockout.js
9. champion.js
10. app.js cleanup

After each move:

Commit
