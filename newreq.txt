Requirement - Match Promotion & Shareable Match Page
Objective

Create a Match Promotion feature that allows administrators to promote an important upcoming match and generate a beautiful, shareable page to increase excitement and community participation.

The goal is not only to inform residents about an upcoming match but to encourage families, children, and community members to attend, support the players, and celebrate sportsmanship together.

This feature complements:

Live Matches
Champions
Event Report

by creating excitement before the match.

Design Philosophy

The application should support the complete event lifecycle.

Administrator Actions
Promote Match

Every eligible match should display:

⭐ Promote Match

When enabled:

Only one match can be promoted at a time.
Previous promoted match is automatically unselected.

Event details

Data Model

Store only minimal promotion metadata.

Example

{
  "matchId": "...",
  "promoted": true,
  "published": true,
  "publishedAt": "...",
  "shareUrl": "..."
}

All other information is derived from the tournament state.

Performance

Promotion page should be generated once.

Viewers only read the published page.

Avoid regenerating content for every request.

Non-Goals

The promotion page is not:

Live scoring page
Match administration page
Statistics dashboard

Its sole purpose is to build anticipation, increase attendance, and create excitement around important matches.

Benefits
Community Engagement

Encourages residents to attend important matches.

Better Atmosphere

Players perform in front of an engaged audience.

Increased Participation

Parents and children become more involved in future events.

Better Communication

Organizers can share one professional link instead of manually typing WhatsApp messages.

Professional Image

Transforms the application from tournament management software into a complete sports event platform.

Recommendation

Treat Match Promotion as a permanent feature of the platform.

Just as the Event Report celebrates what has already happened, the Match Promotion page should build excitement for what is about to happen.

Together they complete the journey of a sporting event:

Promotion → Competition → Celebration → Memory

making the platform valuable not only for organizers but for the entire community.
